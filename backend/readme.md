# Setup

## Clone the backend
0. run cmd `git clone https://github.com/Veer376/backend` & `cd backend` to clone the repo and navigate to the project directory.
1. rename `.env.example` to `.env` and add your Groq API key in the `.env` file.
2. run `npm install` to install dependencies.
3. run `npm run dev` to start the server.


## If backend if already cloned
a. `git pull origin main`
b. `npm i` to install the dependencies
c. `npm run dev` to start the backend
